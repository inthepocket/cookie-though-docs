Configuration

How to configure Cookie Though.

Create a configuration object

To configure Cookie Though, create a configuration object and pass it to the init()init() function.

import { init } from 'cookie-though';

const config = {};
init(config);
import { init } from 'cookie-though';

const config = {};
init(config);

Changing the configuration

If you need to change the configuration at runtime, if you wish to change the language for example, you can do so using the configure()configure() function with the new configuration object.

import { init, configure } from 'cookie-though';

const englishConfig = {};
const dutchConfig = {};

init(englishConfig);

const onLanguageChange = () => {
  configure(dutchConfig);
};
